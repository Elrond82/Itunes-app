# 🎵 iTunes Explorer - React API Project

¡Bienvenido! Este es un buscador de música interactivo que consume la **iTunes Search API**. El proyecto fue desarrollado para practicar el manejo de estados complejos en React, navegación de datos en múltiples niveles y consumo de APIs REST.

## 🚀 Características
- **Búsqueda Dual:** Permite buscar específicamente por nombre de Artista o por título de Álbum de forma independiente.
- **Navegación de 3 Niveles:** 1. Búsqueda de entidad (Artista/Disco).
    2. Selección visual desde una grilla de álbumes.
    3. Detalle completo de canciones y duración.
- **UI Interactiva:** Efectos de hover, scroll automático hacia los resultados y visualización de carátulas en alta resolución.
- **Optimización de Datos:** Conversión de milisegundos a formato de tiempo real (mm:ss).

## 🛠️ Tecnologías Usadas
* **React.js** (Vite)
* **JavaScript (ES6+)**
* **CSS3** (Diseño responsive y efectos visuales)
* **iTunes Search API** (Consumo de datos en tiempo real)
